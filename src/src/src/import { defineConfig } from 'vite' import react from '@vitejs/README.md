## How to Play

1. Choose a category (Vocabulary, Grammar, or Idioms)
2. Select difficulty level (Easy, Medium, or Hard)
3. Answer questions before time runs out
4. Get bonus points for answering quickly
5. Keep your streak going for maximum points!

## Game Rules

- You start with 3 lives (hearts)
- Each wrong answer costs 1 life
- When you run out of lives, the game ends
- Correct answers increase your streak
- Quick answers earn bonus points based on time remaining
- 15 questions per category (5 per difficulty level)

## Scoring System

- **Easy Questions**: 10 points + time bonus
- **Medium Questions**: 20 points + time bonus
- **Hard Questions**: 30 points + time bonus
